sandphoto
=========
prerequisites
```sh
sudo apt-get install php7.2-cli php7.2-gd
```

make biometric photo locally
1. put the photo to the same directory and name as "test.jpg"
2. run
```sh
php sandphoto.inc > 6inch.jpg
```
